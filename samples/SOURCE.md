# Sample Image Source

The reference and target images in this folder were downloaded from:

- Repository: `simo-bat/Crack_detection`
- Folder: `test_images/`
- URL: https://github.com/simo-bat/Crack_detection/tree/main/test_images

The repository README says these images are used as test images for a pavement surface crack detection app based on the SDNET2018 concrete crack dataset.
